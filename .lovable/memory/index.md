# Project Memory

## Core
Stratview One admin panel. Poppins font. Colors: #1b4263 navy, #0d5a5a, #1f7a7a teal, #4fc9ab mint.
Frontend-only with mock data. Backend dev connects PHP Yii2 APIs. No forgot password needed.
Three admin roles: Super Admin (all), Content Admin (Overview/Categories/Datasets/Dashboards/Notifications), Sales Admin (Overview/Leads).
